Test Scenarios

This page is designed to test nested selector scenarios:

  • Same button labels: All cards have "Add to Cart" and "View Details" buttons - requires parent context
  • Same structure: All cards have identical HTML structure - tests selector nesting
  • Different content: Each card has unique product name, price, and description
  • Data attributes: Cards have data-testid, data-category, and data-product-id for testing
  • Badge variations: Different badges (New, Sale, Popular) can be used for parent selectors

Example nested selector scenarios:

  • Click "Add to Cart" button in the "Gaming Laptop Pro" card
  • Click "View Details" for the product with badge "Sale"
  • Click "Add to Cart" in the card with category "audio"
  • Click button in card with product-id "105"
