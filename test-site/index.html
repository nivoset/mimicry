Mimic Test Site

A collection of static HTML pages for testing Mimic's AI-powered browser automation capabilities.

About This Test Site

This test site provides various scenarios to test Mimic's ability to:

  • Navigate between pages and handle browser history
  • Click buttons and links with various selectors
  • Interact with forms (text inputs, selects, checkboxes, radio buttons)
  • Handle dynamic content that loads after page load
  • Work with accessibility features (aria-labels, roles, semantic HTML)
  • Navigate multi-step flows and wizards
  • Interact with complex UI patterns (modals, tabs, accordions, dropdowns)

Variation System

All pages include a variation system that can change content dynamically over time. This helps test Mimic's ability to handle changing content.

The variation system supports:

  • Random variation: Content changes randomly on each page load
  • Time-based variation: Content changes based on time of day or day of week
  • Rotation system: Cycles through predefined variations
  • State persistence: Uses localStorage to track variations

See the assets/js/variation.js file for implementation details.
